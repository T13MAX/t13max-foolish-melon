# 呆瓜空岛插件

### 模块

| 模块              | 名称              | 描述                                |  
|:----------------|:----------------|:----------------------------------|
| [common-plugin] | 一些每个插件都要依赖的基础东东 | 所有插件都会依赖这个包                       |
| [common-data]   | 持久化             | bukkit插件支持存储数据, 自定义的持久化系统有点麻烦暂时搁置 |
| [common-util]   | 工具类             |                                   |
| [plugin-xxx]    | xxx插件           | 未来可能会有多个插件                        |
| [hmcl-client]   | 我的世界客户端         | 内置在项目里, 方便拉取调试                    |
| [spigot-server] | 我的世界服务器         | 内置在项目里, 方便拉取调试                    |
